# AI Chat Assistant

基于 Next.js 14 和 DeepSeek API 构建的智能聊天助手。

## 快速开始

1. 安装依赖：
```bash
npm install
```

2. 创建环境变量文件 `.env.local`：
```bash
OPENAI_API_KEY=your_api_key
```

3. 启动开发服务器：
```bash
npm run dev
```

## 主要功能

- 实时 AI 对话
- Markdown 格式支持
- 代码高亮显示
- 响应式设计
- 移动端适配

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- DeepSeek API
